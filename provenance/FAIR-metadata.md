# FAIR Metadata (Bache Archive Meta)

**Findable** — Persistent identifiers (Wikidata QIDs) stored in `wikidata.jsonld` and mirrored via Git tags and Zenodo DOIs for releases.  
**Accessible** — Public CC0-licensed JSON/JSON-LD/YAML hosted on GitHub; downloadable raw URLs referenced in CI.  
**Interoperable** — JSON-LD context (`wd`) aligns with Linked Open Data; schemas compatible with CSL-JSON, schema.org, SKOS, and PROV-O.  
**Reusable** — CC0 1.0 dedication; provenance tracked in Git history and release notes.

This registry is the canonical source for cross-repository identifiers used by the Bache Archive Project.
